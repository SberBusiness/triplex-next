<Title>v1.24.0</Title>

<Heading>Ломающие изменения</Heading>

**Dropdown**

- Удалено значение `AUTO` из `EDropdownAlignment`, используйте `LEFT` или `RIGHT`.

<Heading>Изменения</Heading>

**Checkbox**

- Исправлен адаптивный вид `CheckboxYGroup`.

**Col**

- Уменьшен горизонтальный отступ между колонками: `@grid-horizontal-gap` изменён с `12px` на `8px`.

**Dropdown**

- Для значений `LEFT`/`RIGHT` из `EDropdownAlignment` внедрено адаптивное позиционирование.

**SelectExtendedField**

- Исправлен баг: `onClose` и `onOpen` больше не вызываются при ре-рендере родительского компонента.
- Исправлен баг: `onClose` больше не вызывается при первоначальном монтировании компонента.

**Typography**

- Добавлен компонент `CodeText` для отображения моноширинного текста (шрифт SBSansTextMono, 14px / 20px).
