<Title>v1.27.0</Title>

<Heading>Изменения</Heading>

**DateField**

- Исправлен `displayName`.

**DateField / MonthYearField**

- Исправлен размер иконки календаря для размера `LG`.

**IslandAccordion**

- Исправлены отступы.
- В адаптиве размер `Title` изменен на `H3`.

**IslandWidget**

- Добавлено свойство `size` (`EComponentSize`) для управления скруглением углов и отступами.
- Область `IslandWidget.Header.Content` заменена на `IslandWidget.Header.Title` со встроенным компонентом типографики и `IslandWidget.Header.Controls`.
- Компонент типографики для `IslandWidget.Header.Description` перенесен внутрь компонента.
- Добавлен перенос текста `IslandWidget.Header.Title` на новую строку в адаптиве.
