# Gap

## Назначение

Вертикальный отступ-разделитель фиксированной высоты между блоками. Рендерит пустой `<div>` с `role="presentation"` без визуального контента — только пространство.

Используй когда: нужно добавить вертикальный отступ между двумя соседними блоками из заранее заданной шкалы значений дизайн-системы (4, 8, 12, 16, 24, 32, 64, 128 px).

Не используй когда:
- Нужен горизонтальный отступ — управляй им через `margin`/`padding` родителя или через `Col`/`Row` сетки.
- Нужна визуальная линия-разделитель — используй `Divider`.
- Нужен отступ произвольной величины (не из шкалы) — задай его через стили родителя, не вводи новое значение `TGapSize`.

---

## Варианты и props

### Обязательные props

| Prop | Тип | Описание |
|---|---|---|
| `size` | `TGapSize` | Высота отступа в пикселях. Один из значений шкалы: `4 \| 8 \| 12 \| 16 \| 24 \| 32 \| 64 \| 128`. |

### Опциональные props

Компонент расширяет `Omit<React.HTMLAttributes<HTMLDivElement>, "children">`, поэтому принимает все стандартные атрибуты `<div>` (включая `className`, `style`, `aria-*`, `data-*`) кроме `children` — пробрасывание содержимого внутрь Gap запрещено типом.

### Ограничения

- `children` исключён из props — Gap не должен оборачивать контент, только разделять его. Если нужно — используй другой контейнер.
- Прокидываемый снаружи `style.height` переопределит высоту, заданную CSS-классом (более высокая специфичность inline-style). Это допустимо для крайних случаев, но штатно высоту задаёт `size`.

---

## Дизайн-токены

Компонент не использует собственных CSS-переменных. Высота задаётся фиксированными значениями в пикселях из CSS-модуля `styles/Gap.module.less` (классы `.size-4`, `.size-8`, ..., `.size-128`).

---

## Инварианты

- **`forwardRef`** — обязателен, не убирать. `ref` пробрасывается на корневой `<div>`.
- **`TGapSize`** — публичный union type, экспортируется из barrel. Изменение перечня значений (добавление/удаление чисел) — breaking change.
- **`IGapProps`** имя интерфейса и `Gap` имя компонента — часть публичного API.
- **`role="presentation"`** на корневом `<div>` — компонент семантически невидим для assistive technologies.
- **`Omit<..., "children">`** в `IGapProps` — публичный контракт, что Gap не принимает потомков.
- **CSS-классы `size-4`...`size-128`** в `Gap.module.less` — могут использоваться в тестах, переименование требует синхронного обновления тестов.

---

## Accessibility

- Корневой `<div>` имеет `role="presentation"` — Gap не несёт смысловой нагрузки и игнорируется screen reader'ами.
- Потребитель может передать дополнительные `aria-*` атрибуты через spread (например, `aria-hidden="true"`), но штатно `role="presentation"` уже исключает компонент из accessibility-дерева.
- Компонент не интерактивный — keyboard navigation и focus management не применимы.

---

## Связанные компоненты

- `Divider` (`src/components/Divider/`) — горизонтальная линия-разделитель с собственными отступами `marginTopSize`/`marginBottomSize`. Используй вместо Gap, когда нужно визуально разделить блоки линией.
- `Col`, `Row` (`src/components/Col/`, `src/components/Row/`) — компоненты сетки. Управляют отступами через систему columns/offsets, а не через явный разделитель.
