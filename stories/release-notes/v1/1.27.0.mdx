<Title>v1.27.0</Title>

<Heading>Изменения</Heading>

**DateField**

- Исправлен `displayName`.

**DateField / MonthYearField**

- Исправлен размер иконки календаря для размера `LG`.

**IslandAccordion**

- Исправлены отступы.
- В адаптиве размер `Title` изменен на `H3`.

**IslandWidget**

- Добавлено свойство `size` (`EComponentSize`) для управления скруглением углов и отступами.

**List**

- Интерфейс `IListItemEmptyProps` переименован в `IListEmptyStateProps` (синхронизация с именем компонента `ListEmptyState`).
- Исправлен `displayName` компонента `ListEmptyState`.

**ListMaster**

- Добавлен `forwardRef` на `ListMaster`. TypeScript-тип компонента изменён с `React.FC<IListMasterProps>` на `ForwardRefExoticComponent<...>` со статическими свойствами (`Body`, `Header`, ...). Удалён интерфейс `IListMasterFC` — для типизации используй `typeof ListMaster`.
- Исправлен `displayName` компонента `ListMasterChipGroup` (был `"ListMasterChips"`).
- Экспортирован интерфейс `IListMasterChipGroupProps` (раньше props инлайнились).
