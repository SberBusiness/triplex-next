<Title>v1.4.0</Title>

<Heading>Изменения</Heading>

**DropdownList**

- Добавлена зависимость максимальной высоты от размера.

<Heading>Новые компоненты</Heading>

**ChipSuggest**

- Chip с выбором одного значения из списка с возможностью фильтрации.

**LightBox/LightBox.SideOverlay**

- Компонент увеличенного модального окна.

**Spoiler**

- Используется для раскрытия внутреннего содержимого.

**SMSInput**

- Компонент для ввода СМС.

**TextareaField**

- Многострочное текстовое поле для ввода большого объема текста.
