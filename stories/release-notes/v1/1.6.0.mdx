<Title>v1.6.0</Title>

<Heading>Новые компоненты</Heading>

**ChipMultiselect**

- Компонент выбора нескольких значений из списка.

**DateRange**

- Компонент выбора периода.

**IslandWidget**

- Компонент-обособленный блок на основе компонента Island, предназначенный для представления сгруппированной информации.

**MultiselectField**

- Компонент мульти-списка.

**TopOverlay**

- Оверлей для предупреждения о незавершенном действии. Используется в LightBox и SideOverlay.

**DocumentNumberEdit**

- Редактируемый номер документа.

**SmallInput**

- Поле ввода для DocumentNumberEdit.

<Heading>Изменения</Heading>

**StatusTracker**

- EVerticalAlign переиенован в EStatusTrackerVerticalAlign.
