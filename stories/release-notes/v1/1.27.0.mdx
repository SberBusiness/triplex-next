<Title>v1.27.0</Title>

<Heading>Ломающие изменения</Heading>

**DateField**

- Теперь кнопка очистки рендерится только при передаче `onClear`.

<Heading>Изменения</Heading>

**DateField**

- Исправлен `displayName`.
- Добавлено свойство `onClear` для обработки клика по кнопке очистки.

**DateField / MonthYearField**

- Исправлен размер иконки календаря для размера `LG`.

**IslandAccordion**

- Исправлены отступы.
- В адаптиве размер `Title` изменен на `H3`.

**IslandWidget**

- Добавлено свойство `size` (`EComponentSize`) для управления скруглением углов и отступами.
