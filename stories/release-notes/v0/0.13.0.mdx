<Title>v0.13.0</Title>

<Heading>Новые компоненты</Heading>

**Avatar**

- Аватар, предназначен для отображения изображений профиля пользователя, инициалов или иконок.

<Heading>Изменения</Heading>

**CheckboxTree, CheckboxTreeExtended**

- Добавлены размеры SM, LG.

**HelpBox**

- Исправлен баг с расположением кнопки закрытия в адаптиве без Header.

**LightBox**

- Добавлено свойство fixed для боковых панелей.

**MultiselectField**

- Добавлено состояние загрузки для MultiselectFieldDropdownContent.

**Tabs**

- Исправлен размер TabButtonDropdown.

**TextField**

- Убран цвет фона при автозаполнении.
- Актуализирована peer-зависимость `@sberbusiness/icons-next`.

**Typography**

- Исправлен баг с отображением шрифтов в разных браузерах.
