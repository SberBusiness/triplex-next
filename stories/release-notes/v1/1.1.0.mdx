<Title>v1.1.0</Title>

<Heading>Изменения</Heading>

**FormField/TextField**
- Удалены свойства disabled/error
- Добавлено свойство status, значениями которого могут быть default/error/disabled/warning

<Heading>Новые компоненты</Heading>

**SelectExtendedField**
- Кастомизируемый компонент выбора значения

**FormFieldTarget**
- Компонент, отображающий нередактируемое значение поля формы

**Island**
- Компонент, реализующий визуальный остров для других компонентов.
