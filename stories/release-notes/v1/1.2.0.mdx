<Title>v1.2.0</Title>

<Heading>Изменения</Heading>

**Island**

- Удалены EIslandBorderRadius и EIslandPaddingSize, вместо них добавлены типы TIslandBorderRadiusSize и TIslandPaddingSize.

**Button**

- Удалены EButtonSize, вместо него добавлен EComponentSize.

**ButtonDropdown**

- Удалены EDropdownSize, вместо него добавлен EComponentSize.

**DropdownDesktop**

- Удалены EDropdownSize, вместо него добавлен EComponentSize.

**DropdownList**

- Удалены EDropdownListSize, вместо него добавлен EComponentSize.

**FormField**

- Удалены EFormFieldSize, вместо него добавлен EComponentSize.

**FormFieldContext**

- Удалены EFormFieldSize, вместо него добавлен EComponentSize.

**FormFieldInput**

- Удалены EFormFieldSize, вместо него добавлен EComponentSize.

**SelectExtendedFieldDropdownDefault**

- Удалены ESelectFieldSize, вместо него добавлен EComponentSize.

<Heading>Новые компоненты</Heading>

**Chip**

- Предоставляет возможность произвести действие по нажатию, также отображает выбранное состояние.

**ChipGroup**

- Контейнер компонентов Chip.

**ChipIcon**

- Chip с иконкой.

**ChipOptions**

- Chip с иконкой выбора опций.

**Stepper**

- Статус-панель с последовательной совокупностью шагов (steps).

**StepperExtended**

- Декларативный вариант Stepper.
