<Title>v1.16.0</Title>

<Heading>Новые компоненты</Heading>

**ExpandAnimation**

- Компонент анимации сворачивания/разворачивания контента.

<Heading>Изменения</Heading>

**Badge**

- Добавлены свойства prefix и postfix.
- Badge в виде точки вынесен в компонент Badge.Dot.

**ChipSort**

- Добавлена возможность взаимодействия с компонентом с помощью клавиатуры;
- Исправлен баг, когда при повторном клике по компоненту дропдаун не закрывался.

**DateField**

– Исправлен баг с передачей postfix.

**FooterPage**

- Исправлена тень, когда Footer прилипает к нижней части экрана.

**HelpBox**

– Добавлено свойство iconProps для изменения цвета иконки.

**LightBoxSideOverlay**

- Удален enum ELightBoxSideOverlaySize. Компонент теперь использует EComponentSize.

**SelectExtendedFieldDropdown**

- Исправлен баг, когда SelectExtendedField.Dropdown.List всегда использовался в размере MD.

**Tabs**

- Исправлены вертикальные отступы в TabButton.
